## Flipper Laser Tag

1) The ".ir" files go in the infrared folder on the Flipper.

2) Navigate to Infrared -> Saved Remotes -> Select YOUR color.

3) Choose one of the weapons: Pistol, Shotgun, Assault, Bazooka. Holding down the Bazooka button will cause your opponent to lose immediately. üòè
